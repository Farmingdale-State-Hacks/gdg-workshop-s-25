About DetectiveCase

A comprehensive crime pattern explorer using Google Cloud to analyze data from New York City police reports.

DetectiveCase is a web application designed to help researchers, students, and citizens explore crime patterns in New York City, especially the changes that occurred during the COVID-19 pandemic. By leveraging BigQuery, Cloud Run, and other Google Cloud services, we're able to analyze over 6 million crime records quickly and efficiently.

Powered by Google Cloud

DetectiveCase leverages multiple Google Cloud services to handle large datasets and scale efficiently.

Cloud Run

Our backend services run on Cloud Run, which automatically scales based on traffic and allows us to focus on building features rather than managing infrastructure.

BigQuery

BigQuery allows us to analyze over 6 million crime records in seconds, making complex queries and pattern detection possible in real-time.

Cloud Storage

We use Cloud Storage to host static assets, exported charts, and cached datasets, ensuring reliable and fast access to resources.
